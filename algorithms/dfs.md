# Depth First Search (DFS)

Definition:

A graph traversal algorithm that explores
as deeply as possible before backtracking.

Uses:

- stack
- recursion

Example:

A
↓
B
↓
C
↑
B
↑
A

Applications:

- connected components
- tree traversal
- file systems
- cycle detection
- graph search

Complexity:

O(V + E)
