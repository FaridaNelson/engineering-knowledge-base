# Connected Components

A connected component is a group of nodes connected together.

Example:

111001
100001
111110

There are two connected groups of zeros.

Algorithm:

1. Find an unvisited node.
2. Increment component count.
3. DFS/BFS all connected nodes.
4. Continue searching.

Applications:

- bitmap holes
- island counting
- social networks
- clustering
- computer vision segmentation
